# De Bruijn Sequence Utilities

A comprehensive Python module for generating and working with De Bruijn sequences.

## What is a De Bruijn Sequence?

A De Bruijn sequence B(k, n) is a cyclic sequence of length k^n in which every possible length-n string on a k-symbol alphabet occurs exactly once as a substring. For example, B(2, 3) = `00010111` contains all 8 possible 3-bit strings: 000, 001, 010, 011, 100, 101, 110, 111.

## Applications

- **Password/PIN Cracking**: Optimize brute-force attacks by testing all combinations with minimal keystrokes
- **DNA Sequencing**: Genome assembly from short reads using k-mer analysis
- **Combinatorial Testing**: Efficient test pattern generation for state machines
- **Memory Testing**: Address pattern generation for memory validation
- **Protocol Testing**: Packet header combination testing
- **Barcode Scanner Testing**: Continuous scanning test patterns

## Features

- Generate De Bruijn sequences for any alphabet size and substring length
- Custom alphabet support
- Sequence validation
- Substring extraction and position finding
- Memory-efficient generator for large sequences
- Convenience functions: binary, decimal, hexadecimal, DNA, alphabet
- Class-based interface for object-oriented usage

## Quick Start

```python
from de_bruijn import binary_de_bruijn, DeBruijnSequence, is_de_bruijn

# Generate a binary sequence
seq = binary_de_bruijn(3)  # '00010111'
print(f"Length: {len(seq)}")  # 8

# Validate
print(is_de_bruijn(seq, 3))  # True

# Use class interface
dbs = DeBruijnSequence(2, 4)
print(dbs.contains('1010'))  # True
print(dbs.position('1111'))  # 12
```

## API Reference

### Core Functions

| Function | Description |
|----------|-------------|
| `de_bruijn(k, n, alphabet)` | Generate De Bruijn sequence B(k, n) |
| `de_bruijn_generator(k, n, alphabet)` | Memory-efficient generator |
| `is_de_bruijn(sequence, n, alphabet)` | Validate a De Bruijn sequence |
| `get_all_substrings(sequence, n)` | Extract all n-length substrings |
| `find_substring_position(sequence, target)` | Find substring position |

### Convenience Functions

| Function | Description |
|----------|-------------|
| `binary_de_bruijn(n)` | Binary sequence B(2, n) |
| `decimal_de_bruijn(n)` | Decimal sequence B(10, n) |
| `hexadecimal_de_bruijn(n)` | Hex sequence B(16, n) |
| `dna_de_bruijn(n)` | DNA sequence B(4, n) with A,C,G,T |
| `alphabet_de_bruijn(n)` | Letter sequence B(26, n) |

### Class Interface

```python
class DeBruijnSequence(k, n, alphabet):
    - sequence: The generated sequence
    - contains(substring): Check if substring exists
    - position(substring): Find position of substring
    - all_substrings(): Get all n-length substrings
    - is_valid(): Validate the sequence
    - rotate(offset): Rotate the sequence
    - complement(): Get complement (binary only)
    - to_numbers(): Convert to numeric list
```
