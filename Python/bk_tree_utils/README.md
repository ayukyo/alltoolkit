# BK-Tree Utils

A pure Python implementation of BK-Tree (Burkhard-Keller Tree) for efficient approximate string matching and spell checking.

## Features

- **Zero External Dependencies** - Pure Python implementation using only standard library
- **Efficient Search** - O(log n) average case for approximate string matching
- **Spell Checking** - Built-in spell checker with suggestions
- **Customizable** - Support for custom distance functions
- **Fully Tested** - Comprehensive unit tests included

## Installation

No installation required. Simply copy the `bk_tree.py` file to your project.

```python
from bk_tree import BKTree, SpellChecker
```

## Quick Start

### Basic Usage

```python
from bk_tree import BKTree

# Create a BK-Tree
tree = BKTree()

# Insert words
words = ["hello", "hallo", "help", "shell", "held"]
for word in words:
    tree.insert(word)

# Search for similar words (within edit distance 1)
results = tree.search("hell", max_distance=1)
print(results)  # ['hello', 'hallo', 'help']

# Find the nearest word
nearest = tree.find_nearest("helo")
print(nearest)  # 'hello'

# Check if word exists
if "hello" in tree:
    print("Found!")
```

### Spell Checker

```python
from bk_tree import SpellChecker

# Create spell checker with dictionary
checker = SpellChecker(["python", "programming", "computer", "algorithm"])

# Check spelling
if checker.is_correct("python"):
    print("Correct!")

# Get suggestions for misspelled word
suggestions = checker.suggest("pythn", max_distance=2, max_suggestions=3)
print(suggestions)  # ['python']
```

### Levenshtein Distance

```python
from bk_tree import levenshtein_distance

# Calculate edit distance between two strings
distance = levenshtein_distance("kitten", "sitting")
print(distance)  # 3
```

## API Reference

### BKTree Class

```python
tree = BKTree(distance_func=None)
```

Create a BK-Tree instance.

**Parameters:**
- `distance_func` (optional): Custom distance function. Defaults to Levenshtein distance.

**Methods:**

| Method | Description |
|--------|-------------|
| `insert(word)` | Insert a word into the tree |
| `search(word, max_distance=1)` | Find all words within max_distance |
| `find_nearest(word, max_distance=None)` | Find the closest word |
| `contains(word)` | Check if word exists |
| `size()` | Get number of words in tree |
| `is_empty()` | Check if tree is empty |
| `clear()` | Remove all words |
| `get_all_words()` | Get all words as a set |
| `get_height()` | Get tree height |

**Operators:**
- `len(tree)` - Number of words
- `word in tree` - Check if word exists

### SpellChecker Class

```python
checker = SpellChecker(words=None)
```

Create a spell checker instance.

**Parameters:**
- `words` (optional): Initial list of valid words

**Methods:**

| Method | Description |
|--------|-------------|
| `add_word(word)` | Add word to dictionary |
| `add_words(words)` | Add multiple words |
| `is_correct(word)` | Check if word is spelled correctly |
| `suggest(word, max_distance=2, max_suggestions=5)` | Get spelling suggestions |

## Performance

BK-Tree provides significant speedup over brute-force search for approximate string matching:

- **Insert**: O(log n) average case
- **Search**: O(log n) average case for small edit distances
- **Space**: O(n)

For a dictionary of 10,000 words, BK-Tree can be 10-100x faster than brute-force search.

## Algorithm

The BK-Tree is a metric tree that organizes words based on their edit distances:

1. **Insert**: Words are placed at a distance from the root equal to their edit distance from the root word
2. **Search**: Uses triangle inequality to prune search space efficiently

For a detailed explanation, see: [BK-Tree Wikipedia](https://en.wikipedia.org/wiki/BK-tree)
